Membangun Web Modern dengan Gemini & Ekosistem Cloudflare

Panduan teknis langkah-demi-langkah dari nol hingga website Anda online dengan domain profesional hanya seharga kopi sachet.

Estimasi Waktu

45 - 60 Menit

Estimasi Biaya

Rp 3.000 (Tahun I)

Level Kesulitan

Pemula - Menengah

1

Fase 1: AI Prompting dengan Gemini

Jangan sekadar meminta "buatkan web". Gunakan framework **Role-Task-Context-Format** untuk hasil maksimal.

Prompt yang Disarankan
"Bertindaklah sebagai Senior Frontend Developer. Buatkan saya sebuah file tunggal HTML yang berisi: 1. Tailwind CSS via CDN. 2. Desain Landing Page Modern untuk Jasa Fotografi. 3. Dark mode toggler. 4. Animasi halus menggunakan AOS.js. 5. Bagian Pricing, Portfolio Grid, dan Footer. Berikan komentar pada setiap blok kode."

Penting:

Setelah Gemini memberikan kode, simpan file tersebut dengan nama index.html di dalam folder baru di komputer Anda.

2

Fase 2: Persiapan Repositori GitHub

Menyimpan kode di GitHub memungkinkan Cloudflare Pages melakukan auto-update setiap kali ada perubahan.

  • A Buat akun di GitHub.com.
  • B Klik tombol "New Repository", beri nama misalnya my-website, set ke Public.
  • C Upload file index.html Anda langsung melalui browser (Upload files) atau gunakan Git CLI:
git init
git add .
git commit -m "Initial commit from Gemini"
git branch -M main
git remote add origin https://github.com/USERNAME/my-website.git
git push -u origin main
3

Fase 3: Eksekusi Domain .my.id (Rp 3.000)

IDWebhost sering mengadakan promo .my.id seharga Rp 3.000 atau Rp 5.000 (belum termasuk PPN 11%).

1. Pencarian

Cari domain di kolom pencarian. Pilih yang harganya 3rb/5rb. Pastikan durasi pilih 1 Tahun (karena promo hanya berlaku di tahun pertama).

2. Lewati Add-on

Saat ditawarkan Hosting, SSL Berbayar, atau Proteksi ID, cukup pilih "No Thanks" atau lewati. Kita akan menggunakan Cloudflare untuk semua itu secara gratis.

3. Aktivasi Akun

Isi data diri dengan benar. Khusus domain .id, terkadang diperlukan verifikasi email atau nomor HP agar domain segera aktif.

4

Fase 4: Setup DNS di Cloudflare

Kita memindahkan "otak" domain Anda dari IDWebhost ke Cloudflare agar performa website lebih kencang.

  1. Daftar/Login di Cloudflare.
  2. Pilih "Add Site" dan masukkan domain Anda (contoh: namaku.my.id).
  3. Pilih paket Free (Gratis).
  4. Cloudflare akan mendeteksi record DNS yang ada. Klik Continue.
  5. Bagian Krusial: Cloudflare akan memberikan 2 Nameserver (Contoh: alice.ns.cloudflare.com).
  6. Buka Panel IDWebhost, cari menu Domain > Nameservers, hapus yang lama dan ganti dengan milik Cloudflare tadi.
  7. Tunggu masa propagasi (biasanya 15 menit - 2 jam).
5

Fase 5: Deploy ke Cloudflare Pages

Ini adalah tempat website Anda akan "tinggal". Gratis selamanya untuk traffic personal.

Hubungkan GitHub

Di dashboard Cloudflare, buka Workers & Pages > Create Application > Pages > Connect to Git.

Konfigurasi Build

Framework Preset

None / Plain HTML

Build Command

(Kosongkan)

Klik Save and Deploy. Website Anda akan online dalam 10 detik di alamat unik seperti my-website.pages.dev.

6

Fase Akhir: Mapping Domain .my.id

Langkah terakhir agar website .pages.dev Anda tampil di domain .my.id Anda.

Setting Custom Domain

Di dalam proyek Pages Anda di Cloudflare, buka tab "Custom Domains" > Klik "Set up a custom domain". Masukkan domain namaku.my.id Anda.

Konfirmasi CNAME

Cloudflare akan bertanya "Activate domain?". Klik Yes. Cloudflare secara otomatis akan membuatkan record CNAME di DNS Anda. Sangat mudah!

Selamat! 🎉

Anda sekarang memiliki website profesional yang ditenagai AI, diamankan oleh Cloudflare, dan dideploy otomatis oleh GitHub.
